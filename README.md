# Lavadero Canino - Registro Diario

App web para registrar turnos diarios en los locales del lavadero canino
(Sarmiento, Güemes y Constitución) y comparar el % de uso de la camioneta
de traslado.

## Archivos

- `index.html` — calendario mensual con carga de datos
- `informe.html` — reportes comparativos por local con gráficos
- `supabase.js` — configuración de conexión a la base de datos

## Stack

- Frontend: HTML + JS puro
- Base de datos: Supabase (Postgres)
- Hosting: Netlify
